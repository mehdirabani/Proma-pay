اجرای نسخه دسکتاپ ویندوز

1. ابتدا سامانه وب را نصب کنید و سرور محلی را اجرا کنید:

```bash
npm run start
```

2. در پنجره دیگر، نسخه دسکتاپ را اجرا کنید:

```bash
npm run electron
```

3. برای ساخت بسته ویندوز، پس از نصب وابستگی‌های نود:

```bash
npm run build
```

اگر نشانی سرور متفاوت است، مقدار `PROMA_PAY_URL` را پیش از اجرای الکترون تنظیم کنید.
