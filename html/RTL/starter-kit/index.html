توسعه پروژه خود را شروع کنید!

شروع با نیازهای سفارشی پروژه خود با استفاده از یک الگوی آماده که بسیار دشوار است مدیریت کوبا ویژگی های مفیدی را برای شروع توسعه پروژه شما فراهم می کند بدون هیچ تلاشی!

  • Cuba Admin به شما این امکان را می دهد که صفحات شروع را با طرح بندی های مختلف دریافت کنید، از طرح بندی مطابق با خود استفاده کنید الزامات سفارشی و فقط نام تجاری، منو و محتوا را تغییر دهید.

  • همه مؤلفه‌ها در Cuba Admin جدا شده‌اند، یعنی فقط از مؤلفه‌هایی استفاده کنید که واقعاً نیاز دارید! کدهای غیر ضروری و اضافی را به راحتی فقط با حذف مسیر به فایل SCSS، JS خاص حذف کنید.

  • از PUG به عنوان موتور الگو برای تولید سریع صفحات و کل قالب با استفاده از node js استفاده می کند. صرفه جویی زمان خود را برای انجام تغییرات رایج برای هر صفحه (مانند منو، نام تجاری و پاورقی) توسط ایجاد الگو با میناکاری.

کیت شروع چیست؟

کیت شروع مجموعه ای از صفحات با طرح بندی های مختلف است که برای شروع پروژه بعدی شما مفید است فرآیند توسعه از ابتدا بدون زمان.

  • هر طرح‌بندی فقط شامل اجزای اصلی است.

  • طرح مورد نظر خود را از کیت شروع انتخاب کنید، آن را با تغییرات اختیاری مانند رنگ ها سفارشی کنید و نام تجاری، فقط وابستگی لازم را اضافه کنید.

  • استفاده از موتور قالب برای تولید سریع الگوی کامل با چیدمان انتخابی شما و موارد دیگر تغییرات سفارشی

چگونه از کیت استارتر استفاده کنیم؟

HTML

اگر فقط HTML می دانید، طرح مورد نظر خود را از پوشه کیت شروع انتخاب کنید، آن را با تغییرات اختیاری مانند رنگ ها و نام تجاری، فقط وابستگی لازم را اضافه کنید.

PUG

برای استفاده از PUG به node.js و دانش اولیه استفاده از آن نیاز است. استفاده از PUG به عنوان موتور الگو برای با چیدمان انتخابی خود و سایر تغییرات سفارشی به سرعت کل قالب را ایجاد کنید. برای شروع با استفاده از PUG و فرآیند تولید الگو، لطفاً به اسناد الگو مراجعه کنید.
